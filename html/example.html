单词：example     [iɡˈzɑ:mpl]

翻译：n. 例子，实例；模范，榜样     释义：an item of information that is typical of a class or group;
近义词：
  • case
  • illustration
  • instance
  • representative
  • sample
反义词：
  • beckon
形近词：
  • ample
例句:
  1. It is a classic example of how not to design a new city centre.

    这对设计新市中心是个最佳的反面教材。

  2. This church is a classic example of medieval architecture.

    这座教堂是中世纪建筑风格的典型实例。

  3. Her diligence has set an example to the others.

    她的勤奋为其他人树立了榜样。

  4. Comrade LeiFeng set us a good example to emulate.

    雷峰同志为我们树立了可供效仿的榜样。

  5. I must set an example to this young friend.

    我一定要给小朋友做个榜样。

  6. Many great men have risen from poverty---Lincoln, for example.

    许多伟人从贫困中崛起，例如林肯。

  7. This dictionary has many examples of how words are used.

    这部词典有许多说明词语用法的实例。

  8. Can you give an example of how you would do that?

    你能否举例说明你将怎样去做?

  9. Let this be an example to you.

    你要以此为鉴戒。

  10. He can set a good example for Chinese coaches.

    他可以为中国教练做好的示范。

  11. In fact Ukraine offers them a proud example.

    可实施上乌克兰给俄罗斯做了很好的示范。
