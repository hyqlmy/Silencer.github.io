单词：already     [ɔ:lˈredi]

翻译：adv. 已，已经，早已     释义：prior to a specified or implied time;
近义词：
  • beforehand
  • previously
反义词：
  • yet
形近词：
  • ready
例句:
  1. I've already paid my membership dues.

    我已经交了我的会员费。

  2. The new rules have already come into play.

    新规则已经开始生效。

  3. You had already gone when I arrived there.

    当我到达时，你已经走了。

  4. I've been there already and don't want to go again.

    我曾到过那，不想再去了。
